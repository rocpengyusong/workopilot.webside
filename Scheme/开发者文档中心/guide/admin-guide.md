# 管理员操作指南

本指南面向系统管理员和设备管理人员，介绍喔壳智能终端的初始化配置、设备管理和维护等操作。

## 终端初始化

### 首次启动配置

首次使用喔壳智能终端时，需要完成以下初始化步骤：

#### 步骤 1: 网络配置
1. 连接电源，启动终端设备
2. 进入网络设置界面
3. 选择网络连接方式：
   - **有线网络**: 连接以太网线，自动获取IP地址
   - **无线网络**: 选择Wi-Fi网络，输入密码
4. 测试网络连接是否正常

#### 步骤 2: 设备注册
1. 打开智能体客户端应用
2. 扫描设备二维码（或手动输入设备序列号）
3. 登录喔壳平台账号
4. 完成设备绑定
5. 等待服务器验证

#### 步骤 3: 基本设置
1. **语言设置**: 选择系统语言（中文/English）
2. **时区设置**: 选择所在时区
3. **显示设置**: 调整屏幕亮度和显示参数
4. **音量设置**: 调整系统音量

#### 步骤 4: 功能配置
1. 选择启用功能模块：
   - 智能会议
   - 单据处理
   - 语音助手
2. 配置相关参数
3. 同步初始数据

#### 步骤 5: 测试验证
1. 检查网络连接状态
2. 测试语音输入输出
3. 测试摄像头功能
4. 验证与服务器的数据同步

### 自动初始化

终端支持自动初始化功能：
- 插入网络连接后自动获取IP地址
- 检测到服务器后自动完成注册
- 下载最新配置和系统更新
- 完成初始化后进入工作状态

### 批量初始化

对于多台设备的部署，管理员可以：

#### 预配置准备
1. 准备网络配置文件
2. 导入设备序列号清单
3. 设置默认配置模板
4. 准备批量授权文件

#### 批量部署流程
1. 设备上电并连接网络
2. 系统自动识别并下载配置
3. 应用统一的配置模板
4. 批量完成设备注册
5. 生成部署报告

## 设备管理

### 设备注册管理

#### 单台设备注册
1. 登录管理后台
2. 进入"设备管理"页面
3. 点击"添加设备"
4. 输入设备信息：
   - 设备名称
   - 设备序列号
   - 所属部门
   - 安装位置
5. 完成注册

#### 批量设备导入
1. 准备设备清单CSV文件
2. 格式要求：
   ```
   设备名称,序列号,部门,位置
   会议室A-K2,SN2024001,销售部,3楼
   会议室B-K2,SN2024002,技术部,5楼
   ```
3. 在管理后台选择"批量导入"
4. 上传CSV文件
5. 系统自动验证并导入

## 故障处理

### 常见问题

#### 网络问题
**问题**：设备无法连接网络
**排查步骤**：
1. 检查网络连接状态
2. 测试网络连通性
3. 检查IP地址获取
4. 验证DNS配置
5. 检查防火墙设置

**解决方案**：
- 重启网络服务
- 重新配置网络参数
- 更换网络连接方式
- 联系网络管理员

#### 应用问题
**问题**：应用无法启动或崩溃
**排查步骤**：
1. 查看应用日志
2. 检查应用版本
3. 验证配置文件
4. 检查存储空间

**解决方案**：
- 清理应用缓存
- 重新安装应用
- 更新应用版本
- 恢复出厂设置

#### 性能问题
**问题**：设备运行缓慢
**排查步骤**：
1. 检查CPU使用率
2. 查看内存占用
3. 检查存储空间
4. 分析运行进程

**解决方案**：
- 关闭不必要的应用
- 清理系统缓存
- 重启设备
- 升级硬件配置
