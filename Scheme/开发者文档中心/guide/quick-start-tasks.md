# 快速入门任务

本章节通过具体任务帮助您快速熟悉喔壳平台的核心功能。

## 任务一：体验智能审核

### 目标
了解并体验智能审核功能的基本流程。

### 步骤
1. 登录平台
2. 进入审核工作台
3. 查看待审核单据
4. 使用AI助手进行审核
5. 查看审核结果

### 预期结果
- 了解单据审核的基本流程
- 熟悉AI助手的交互方式
- 掌握审核结果的处理方法

## 任务二：配置简单规则

### 目标
学习如何配置基础的审核规则。

### 步骤
1. 进入规则配置中心
2. 创建新规则
3. 设置规则条件
4. 启用规则
5. 测试规则效果

### 预期结果
- 能够创建简单的审核规则
- 理解规则的触发机制
- 掌握规则测试方法

## 任务三：使用数字员工

### 目标
了解数字员工的基本使用方法。

### 步骤
1. 查看可用的数字员工
2. 分配任务给数字员工
3. 监控任务执行状态
4. 查看执行结果

### 预期结果
- 了解数字员工的功能
- 掌握任务分配方法
- 熟悉结果查看流程
