# 平台概览

**AI-Audit Platform** 是一款面向大型企业的下一代智能财务合规审核系统。它突破了传统 OCR+规则引擎的局限，利用 **LLM (大语言模型)** 和 **Agent (智能体)** 技术，实现了从“形式审核”到“实质审核”的跨越。

## 核心痛点解决

在传统的财务共享中心模式下，我们解决了以下核心问题：

::: info 🚫 割裂的审核体验
**旧模式**：审核员需要在影像系统、ERP、OA 之间来回切换屏幕。
**新模式**：**浏览器插件**直接寄生于业务系统，审核建议“召之即来”，无需切换窗口。
:::

::: info 📉 僵化的规则维护
**旧模式**：修改一条“住宿费限额”规则需要 IT 开发介入，周期长。
**新模式**：**自然语言编排**规则，财务专家可直接通过 Prompt 调整逻辑，秒级生效。
:::

::: info 🧩 难以处理非结构化数据
**旧模式**：只能比对数字，无法理解“业务招待费”备注里的合规性风险。
**新模式**：**多模态 AI** 能读懂附件图片、理解审批流备注，并结合制度文档进行语义判断。
:::

## 系统架构

平台采用 **“端-云-脑”** 分层架构设计：

- **端 (Client)**: 浏览器插件 (Sidekick)，负责环境感知与交互。
- **云 (Core)**: Java 微服务，负责业务网关与数据持久化。
- **脑 (Brain)**: .NET Agent 引擎，负责多模态感知、推理与决策。
