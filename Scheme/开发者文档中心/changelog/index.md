# AI智能体平台 - 更新日志

本文档记录了 Danduola Agent 智能审核平台的所有版本更新历史。

## v1.2.0 (2025-12-01)

::: tip 里程碑版本
本次更新引入了全新的混合制度引擎，支持从 PDF 制度文档中自动检索依据。
:::

### ✨ 新增功能
- **混合制度引擎 (Hybrid Policy Engine)**: 
  - 支持上传 PDF/Word 格式的企业财务制度文档。
  - 新增 RAG (检索增强生成) 模块，实现非结构化制度的语义检索。
- **多模态发票比对**: 新增对发票盖章区域的印章识别功能，支持检测印章模糊、遮挡等风险。
- **Webhooks 集成**: 支持在审核流程结束后通过 Webhook 回调外部系统。

### 🚀 优化与改进
- **性能提升**: 规则引擎冷启动时间减少 40%。
- **交互优化**: 优化了浏览器插件在 SAP 系统中的弹窗位置，避免遮挡关键输入框。
- **日志审计**: 增强了操作日志的记录粒度，现在可以追踪每一条 AI 决策的 Token 消耗。

### 🐛 问题修复
- 修复了在低分辨率屏幕下，侧边栏无法完全展开的问题。
- 修复了某些特殊格式的 PDF 发票解析失败的问题。

---

## v1.1.0 (2025-11-15)

### ✨ 新增功能
- **Prompt 调试器**: 策略编排中心新增在线 Prompt 调试工具，支持实时预览 AI 输出。
- **自定义风险等级**: 支持用户自定义风险等级的颜色和标签（如：高风险、中风险、低风险）。

### 🚀 优化与改进
- **模型升级**: 底层模型升级至 GPT-4o，逻辑推理能力显著提升。
- **国际化**: 新增对英文界面的完全支持。

---

## v1.0.0 (2025-10-01)

🎉 **Danduola Agent 正式发布！**

### 核心特性
- **伴随式审核插件**: 支持 Chrome/Edge 浏览器，无缝集成主流 ERP 系统。
- **智能规则编排**: 基于自然语言的规则配置系统。
- **全量凭证扫描**: 支持发票、合同、银行回单等多种凭证的 OCR 识别。
